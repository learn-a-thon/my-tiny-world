## 🍪 기능

* VOD 검색
* VOD 상세 정보 제공
    * 해당 VOD를 시청할 수 있는 플랫폼 정보를 제공
* OTT 서비스 비교
    * 넷플릭스
    * 왓챠
    * 디즈니 플러스
    * 쿠팡 플레이
    * 웨이브
    * 티빙
* 보고싶은 VOD 저장
    * 저장된 목록에 따른 OTT 플랫폼 추천
* 현재 구독중인 OTT 정보
    * 함께 구독중인 친구들 모아보기
